# Style Trainer  
A user friendly tool for fine-tuning the output of Pythia 2.8b to unique styles using LoRA.  

## Purpose  

The purpose of this tool is to provide a simplified means of fine-tuning a pretrained LLM's output style. This requires little more than a basic understanding of command line and using python.

## Quickstart  

Open powershell and navigate to the folder containing this tool and corpus files. Enter ./style_trainer.p1 and follow the prompts to begin or resume training.
